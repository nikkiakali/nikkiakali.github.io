Thinking & Notes

Designing Reliable LLM Agents for Customer Support

Lessons learned from building production support agents: how careful prompt structuring, tool design, and error handling separate functional prototypes from reliable systems. Explores the tradeoffs between agent autonomy and predictability, and why failure modes matter as much as success metrics.

Structuring Job-Application Data from Messy Email Streams

Thoughts on extracting signal from unstructured application workflows. Using domain heuristics, status classification, and strategic LLM calls to transform chaotic email threads into structured data. When to lean on rules-based systems versus when to trust language models.

AI for Safer Dining with Celiac Disease

Exploring how symptom logging, embeddings, and semantic search can reduce risk for people with celiac disease. The challenge of building trust in AI recommendations when mistakes have real health consequences. Combining structured data with LLM-based reasoning to surface patterns humans might miss.

Building Lightweight ML Tools for Personal Use

Why building tools for yourself is one of the best ways to learn. Reflections on implementing transformers from scratch, designing chrome extensions, and integrating OpenAI APIs—small projects that teach systems thinking, API design, and what it takes to ship something functional.
